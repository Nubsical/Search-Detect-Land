# Vision-Based Autnomous Drone Landing 

A project to build a drone that detects a landing target using comptuer vision and lands on it autonomously using ArduPilot

## Goal
The goal of this project is to use onboard cameras and a Raspberry Pi companion computer to detect a target object or AprilTag, estimate its position relative to the drone, and guide the drone to land on it.
